# Text-Analyzer-System
An intelligent text analysis system using F# functional programming, focusing on immutable data structures, functional sequences, and pattern matching. It process text input (files or manual), calculate various statistics, generate reports, and allow users to view results via a simple UI.



## Installation

1. **Install .NET SDK**
   - Download and install the latest [.NET SDK](https://dotnet.microsoft.com/en-us/download/dotnet)
   - Verify installation:
     ```bash
     dotnet --version
     ```

2. **Install VS Code (if not already installed)**
   - Download from [VS Code](https://code.visualstudio.com/)

3. **Install Ionide F# Extension**
   - Open VS Code
   - Go to Extensions (Ctrl+Shift+X)
   - Search for `Ionide-fsharp` and install it
   - This enables F# language support, IntelliSense, and project management

4. **Clone the Repository**
   ```bash
   git clone https://github.com/FatmaAMR/Text-Analyzer-System
   cd TextAnalyzerSystem 

5. **Open the Project in VS Code**
```bash
code .
```

6. **Restore Dependencies and Build**
```bash
dotnet restore
dotnet build
```

7. **Run the Project**

```bash
dotnet run
```
